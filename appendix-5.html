Exercise19
Answer

The graph of a cubic polynomial with a positive lead coefficient will have the same end behavior as the basic cubic, and a cubic with a negative lead coefficient will have the opposite end behavior. Each graph of a cubic polynomial has one, two, or three \(x\)-intercepts, it has two, one or no turning point, and it has exactly one inflection point.

Exercise61
Answer

  1. If \(P(x)\) is a nonconstant polynomial with real coefficients and \(a\) is any real number, then there exist unique polynomials \(q(x)\) and \(r (x)\) such that \begin{equation*}P(x) = (x â a)q(x) + r (x)\end{equation*} where \(\text{deg }r (x)\lt \text{deg }(x â a)\).

  2. Zero

  3. \(P(a) = (a â a)q(a) + r (a) = r (a)\). Because \(\text{deg }r(x) = 0 \), \(r(x) \) is a constant. That constant value is \(P(a) \), so \(P(x) = (x â a)q(x) + P(a)\).

Exercise63
Answer

  1. From the remainder theorem, \( \begin{aligned} P(x) \amp = (x â a)Q(x) + P(a)\\ \amp = (x â a)Q(x) + 0\\ \amp =(x â a)Q(x) \end{aligned} \)

  2. By definition of a factor, if \(x-a\) is a factor of \(P(x) \), then \(P(x) = (x â a)q(x)\), so \(P(x) =(x â a)q(x) + 0\). The uniqueness guaranteed in the remainder theorem tells us that \(P(a) = 0\).
