To Write a Number in Scientific Notation:

  1. Locate the decimal point so that there is exactly one nonzero digit to its left.

  2. Count the number of places you moved the decimal point: This determines the power of \(10\text{.}\)

    1. If the original number is greater than \(10\text{,}\) the exponent is positive.

    2. If the original number is less than \(1\text{,}\) the exponent is negative.
