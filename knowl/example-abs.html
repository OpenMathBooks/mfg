Example2.2.2

Simplify each expression.

  1. \(\abs{3 − 8}\)
  2. \(\abs{3} − \abs{8}\)
Solution
  1. Simplify the expression inside the absolute value bars first.
    \begin{equation*}\abs{3 − 8} = \abs{−5} = 5\end{equation*}
  2. Simplify each absolute value; then subtract.
    \begin{equation*}\abs{3} − \abs{8} = 3 − 8 = −5\end{equation*}
