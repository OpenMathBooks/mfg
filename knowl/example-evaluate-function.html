Example1.2.22

The function \(H\) is defined by \(H=f(s) = \frac{\sqrt{s+3}}{s}\). Evaluate the function at the following values.

  1. \(s=6\)

  2. \(s=-1\)

Solution

  1. \(f(\alert{6})=\frac{\sqrt{\alert{6}+3}}{\alert{6}}= \frac{\sqrt{9}}{6}=\frac{3}{6}=\frac{1}{2}\). Thus, \(f(6)=\frac{1}{2}\).

  2. \(f(\alert{-1})=\frac{\sqrt{\alert{-1}+3}}{\alert{-1}}= \frac{\sqrt{2}}{-1}=-\sqrt{2}\). Thus, \(f(-1)=-\sqrt{2}\).
