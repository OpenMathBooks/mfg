Example1.1.27
  1. Write a piecewise definition for \(g(x) = \abs{x − 3}\).
  2. Sketch a graph of \(g(x) = \abs{x − 3}\).
Solution
  1. In the definition for \(\abs{x}\), we replace \(x\) by \(x − 3\) to get
    \begin{equation*} g(x) =\abs{x-3} = \begin{cases} x-3 \amp \text{if }~ x-3\ge 0\\ -x \amp \text{if }~ -(x-3)\lt 0 \end{cases} \end{equation*}
    We can simplify this expression to
    \begin{equation*} g(x) =\abs{x-3} = \begin{cases} x-3 \amp \text{if }~ x\ge 3\\ -x+3 \amp \text{if }~ x\lt 3 \end{cases} \end{equation*}
  2. In the first region, \(x \ge 3\), we graph the line \(y = x − 3\). Because \(x = 3\) is included in this region, the endpoint of this portion of the graph, \((3, 0)\), is included, too. In the second region, \(x \lt 3\), we graph the line \(y = −x + 3\). Note that the two pieces of the graph meet at the point \((0, 3)\), as shown in Figure 1.1.28.
