  2. Compute the ratio of each function value to the previous one. Explain the result.

Answer

  1. \(x\) \(0\) \(0.5\) \(1\) \(1.5\) \(2\) \(2.5\)
    \(e^x\) \(1 \) \(1.6487\) \(2.7183\) \(4.4817\) \(7.3891\) \(12.1825\)

  2. Each ratio is \(e^{0.5} \approx 1.6487\): Increasing \(x\)-values by a constant \(\Delta x = 0.5\) corresponds to multiplying the \(y\)-values of the exponential function by a constant factor of \(e^{\Delta x}\).
