  2. Each difference in \(x\)-values is approximately \(\ln 2\approx 0.6931\): Increasing \(x\)-values by a constant \(\Delta x = \ln 2\) corresponds to multiplying the \(y\)-values of the exponential function by a constant factor of \(e^{\Delta x} = e^{\ln 2} = 2\). That is, each function value is approximately equal to double the previous one.
