31

  1. Delbert reads the following values from the graph of a function: \begin{equation*}f (−3) = 5, ~f (−1) = 2, ~f (1) = 0,\end{equation*} \begin{equation*}f (−1) = −4, ~f (−3) = −2\end{equation*} Can his readings be correct? Explain why or why not.

  2. Francine reads the following values from the graph of a function: \begin{equation*}g(−2) = 6, ~g(0) = 0, ~g(2) = 6,\end{equation*} \begin{equation*}g(4) = 0, ~g(6) = 6\end{equation*} Can her readings be correct? Explain why or why not.

Answer

  1. He has an error: \(f (−3)\) cannot have both the value \(5\) and also the value \(−2\) and \(f (−1)\) cannot have both the value of \(2\) and \(-4\).

  2. Her readings are possible for a function: each input has only one output.
