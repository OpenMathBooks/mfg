Graph \(y = 6.5 − 1.8x\).

  1. For what value of \(x\) is \(y = -13.3\)?

  2. For what value of \(x\) is \(y = 24.5\)?

  3. For what value of \(x\) is \(y\le 15.5 \)?

  4. For what value of \(x\) is \(y\ge −7.9\)?

Answer

  1. \(x = 11\)

  2. \(x = −10\)

  3. \(x\ge -5\)

  4. \(x\le 8 \)
