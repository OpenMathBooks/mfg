Exercise15

Let \(f(x) = 3^x\) and \(g(x) = \log_3 x\).

  1. Compute \(f (4)\).

  2. Compute \(g[ f (4)]\).

  3. Explain why \(\log_3 3^x = x\) for any \(x\).

  4. Compute \(\log_3 3^{1.8}\).

  5. Simplify \(\log_3 3^a\).

Answer

  1. \(81\)

  2. \(4\)

  3. Definition of logarithm base \(3\)

  4. \(1.8\)

  5. \(a\)
