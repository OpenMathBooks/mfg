Exercise15

  1. Find \(f^{−1}\) for the function \(f (x) = (x − 2)^3\).

  2. Show that \(f^{−1}\) undoes the effect of \(f\) on \(x = 4\).

  3. Show that \(f\) undoes the effect of \(f^{−1}\) on \(x = -8\).

  4. Graph the function and its inverse on the same grid, along with the graph of \(y = x\).

Answer

  1. \(f^{−1} (y) = 3 \sqrt[3]{y} + 2\)

  2. \(f^{−1} ( f (4)) = f^{−1} (8) = 4\)

  3. \(f ( f^{−1} (−8)) = f (0) = −8\)
