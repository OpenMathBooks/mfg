An electrical component of a high-tech sensor requires \(0.25\) ounces of gold. Assume that the actual amount of gold used, \(g\), is not in error by more than \(0.001\) ounce. Write an absolute value inequality for the possible error and show the possible values of g on a number line.

Answer

\(\abs{g-0.25}\le 0.001\)
