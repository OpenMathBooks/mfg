Exercise33

  1. Find the domain and range of the function \(g(x) = \sqrt{4 − x}\).

  2. Find a formula for \(g^{−1}(x)\).

  3. State the domain and range of \(g^{−1}(x)\).

  4. Graph \(g\) and \(g^{−1}\) on the same grid.

Answer

  1. Domain: \((−\infty, 4]\); Range: \([0,\infty)\)

  2. \(g^{-1}(x)=4-x^2 \)

  3. Domain: \([0,\infty)\); Range: \((-\infty,4]\)
