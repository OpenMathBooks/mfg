Exercise39

River Queen Tours offers a \(50\)-mile round-trip excursion on the Mississippi River on a paddle wheel boat. The current in the Mississippi is \(8\) miles per hour.

  1. Express the time required for the downstream journey as a function of the speed of the paddle wheel boat in still water.

  2. Write a function for the time required for the return trip upstream.

  3. Write and simplify an expression for the time needed for the round trip as a function of the boat's speed.

Answer

  1. \(\dfrac{25}{s+8} \)

  2. \(\dfrac{25}{s-8} \)

  3. \(\dfrac{50s}{s^2-64} \)
