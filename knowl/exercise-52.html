Suppose that the reduced row echelon form of an augmented matrix is \begin{equation*} \left[ \begin{array}{@{}cccc|c@{}} 1 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \end{array}\right] \end{equation*}

  1. To what equation does the last row of the matrix correspond?

  2. Is there a solution to the system?

Answer

  1. \(0=1\)

  2. No
