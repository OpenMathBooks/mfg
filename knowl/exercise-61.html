Exercise 1.28
Decide whether each table describes \(y\) as a function of \(x\). Explain your choice.
  1. \(x\) \(3.5\) \(2.0\) \(2.5\) \(3.5\) \(2.5\) \(4.0\) \(2.5\) \(3.0\)
    \(y\) \(2.5\) \(3.0\) \(2.5\) \(4.0\) \(3.5\) \(4.0\) \(2.0\) \(2.5\)
  2. \(x\) \(-3\) \(-2\) \(-1\) \(0\) \(1\) \(2\) \(3\)
    \(y\) \(17\) \(3\) \(0\) \(-1\) \(0\) \(3\) \(17\)
Answer

  1. No, for example, \(x = 3.5\) corresponds both to \(y = 2.5\) and also to \(y = 4.0\).

  2. Yes, each value of \(x\) has exactly one value of \(y\) associated with it.
