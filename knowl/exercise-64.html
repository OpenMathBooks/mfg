Exercise57

  1. Show that \(x + x^{−1} = \dfrac{x^2 +1}{x}\).

  2. Show that \(x^3 + x^{−3} = \dfrac{x^6 +1}{x^3}\).

  3. Write \(x^n + x^{−n}\) as an algebraic fraction. Justify your answer.

Answer

  1. \(x + x^{−1} = x + \dfrac{1}{x} = \dfrac{x^2}{x}+\dfrac{1}{x}= \dfrac{x^2 + 1}{x}\)

  2. \(x^3 + x^{−3} = x^3 + \dfrac{1}{x^3} = \dfrac{x^6}{x^3}+\dfrac{1}{x^3}= \dfrac{x^6 + 1}{x^3}\)

  3. \(x^n + x^{−n} = x^n + \dfrac{1}{x^n} = \dfrac{x^{2n}}{x^n}+\dfrac{1}{x^n}= \dfrac{x^{2n} + 1}{x^2}\)
