Verify the following products discussed in the text.

19

\((x + y)^3 = x^3 + 3x^2 y + 3xy^2 + y^3\)

Answer

\( \begin{aligned} (x + y)^3 \amp = (x + y)(x + y)^2\\ \amp = (x + y)(x^2 + 2xy + y^2) \\ \amp = x^3 + 2x^2 y + xy^2 + x^2 y + 2xy^2 + y^3\\ \amp = x^3 + 3x^2 y + 3xy^2 + y^3 \end{aligned} \)
20

\((x - y)^3 = x^3 - 3x^2 y + 3xy^2 - y^3\)

21

\((x + y)(x^2 − xy + y^2) = x^3 + y^3\)

Answer

\( \begin{aligned} (x + y)(x^2 − xy + y^2) \amp = x^3 -x^2 y + xy^2 + x^2 y -xy^2 + y^3\\ \amp = x^3 + y^3 \end{aligned} \)
