For Problems 53–56, graph the function and answer the questions.

53

\(f (x) = \abs{x + 4} + \abs{x − 4}\)

  1. Using your graph, write a piecewise formula for \(f (x)\).

  2. Experiment by graphing \(g(x) = \abs{x + p} + \abs{x − q}\) for different positive values of \(p\) and \(q\). Make a conjecture about how the graph depends on \(p\) and \(q\).

  3. Write a piecewise formula for \(g(x) = \abs{x + p} + \abs{x − q}\).

54

\(f (x) = \abs{x + 4} - \abs{x − 4}\)

  1. Using your graph, write a piecewise formula for \(f (x)\).

  2. Experiment by graphing \(g(x) = \abs{x + p} - \abs{x − q}\) for different positive values of \(p\) and \(q\). Make a conjecture about how the graph depends on \(p\) and \(q\).

55

\(f (x) = \abs{x + 4} +\abs{x} + \abs{x − 4}\)

  1. Using your graph, write a piecewise formula for \(f (x)\).

  2. What is the minimum value of \(f (x)\)?

  3. If \(p, q \ge 0\), what is the minimum value of \(g(x) = \abs{x + p}+\abs{x} + \abs{x − q}\)?

56
