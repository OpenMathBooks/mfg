State the domain and range of each basic function.

9

  1. \(f(x)=x^3\)

  2. \(g(x)=x^2\)

Answer

  1. Domain: all real numbers; Range: all real numbers

  2. Domain: all real numbers; Range: \([0, \infty)\)
