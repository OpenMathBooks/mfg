In Problems 25–28, graph each set of functions in the standard window.What do you notice about the \(x\)-intercepts? Generalize your observation, and test your idea with examples.

25

  1. \(y = x^2 − x − 20\)

  2. \(y = 2(x^2 − x − 20)\)

  3. \(y = 0.5(x^2 − x − 20)\)

Answer

The 3 graphs have the same \(x\)-intercepts. In general, the graph of \(y = ax^2 + bx + c\) has the same \(x\)-intercepts as the graph of \(y = k(ax^2 + bx + c)\).
