In Problems 11–14, show how to use the graphs to find the values. Estimate your answers to one decimal point. Compare your estimates to values obtained with a calculator.

11

Refer to the graph of \(f (x) = x^3\).

  1. Estimate the value of \((1.4)^3\).

  2. Find all numbers whose cube is \(−20\).

  3. Find all solutions of the equation \(x^3 = 6\).

  4. Estimate the value of \(\sqrt[3]{24} \).

12

Refer to the graph of \(f (x) = x^2\).

  1. Estimate the value of \((-2.5)^2\).

  2. Find all numbers whose square is \(12\).

  3. Find all solutions of the equation \(x^2 = 15\).

  4. Estimate the value of \(\sqrt{10.5} \).

13

Refer to the graph of \(f (x) = \dfrac{1}{x} \).

  1. Estimate the value of \(\dfrac{1}{3.4}\).

  2. Find all numbers whose reciprocal is \(-2.5\).

  3. Find all solutions of the equation \(\dfrac{1}{x} = 4.8\).

14

Refer to the graph of \(f (x) =\abs{x-2} \).

  1. Estimate the value of \(\abs{1.6-2} \).

  2. Find all values of \(x\) for which \(\abs{x-2}=3\).

  3. Find all solutions of the equation \(\abs{x-2} = 0.4\).
