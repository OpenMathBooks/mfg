  • Exponential Notation

    The absolute value has the following properties: \begin{alignat*}{2} \amp a^{-n}=\frac{1}{x^n} \amp\hphantom{00000} \amp a\ne 0\\ \amp a^{0}=1 \amp \amp a\ne 0\\ \amp a^{1/n}=\sqrt[n]{a} \amp \amp n \text{ an integer, } n\gt 2\\ \amp a^{m/n}=(a^{1/n})^m=(a^m)^{1/n}, \amp \amp a\gt 0, ~n\ne 0 \end{alignat*}
