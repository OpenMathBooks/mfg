Item c

We must solve the equation \begin{equation*}x^3 − 24x^2 + 195x + 250 = 1200\end{equation*} We will solve the equation graphically, as shown in Figure 7.1.14b. Graph \(y = 1200\) along with the cost function, and use the intersect command to find the intersection point of the graphs, \((15.319, 1200)\). \(C(x) = 1200\) when \(x\) is about \(15.319\), so \(15,319\) copies can be printed for \(\$1200\).
