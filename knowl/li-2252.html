To find the inverse function, we solve for \(x\). Take the reciprocal of both sides of the equation. \begin{align*} \frac{1}{y} \amp= x + 3\amp\amp\text{Subtract 3 from both sides.}\\ x \amp = \frac{1}{y}− 3 \end{align*} The inverse function is \(x = f^{−1}(y) = \frac{1}{y}− 3\), or, using \(x\) for the input variable, \(f^{−1}(x) = \frac{1}{x}− 3\). The graph of \(f^{−1}\) looks like the graph of \(y = \frac{1}{x}\), shifted down \(3\) units, as shown in Figure 5.36.
