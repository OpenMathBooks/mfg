On a piano, the frequency of the highest note is about \(4200\) hertz. This frequency is \(150\) times the frequency of the lowest note, at about \(28\) hertz. Ideally, only the lengths of the strings should change, so that all the notes have the same tonal quality. If the string for the highest note is \(5\) centimeters long, how long should the string for the lowest note be?
