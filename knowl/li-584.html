Item c
Using the row for width \(3\) cm, we make a new table showing the relationship between load and depth. The increase in load for each increase of \(1\) cm in depth is not a constant, so the graph in Figure 3.5.6 is not a straight line. The curve does pass through the origin, so perhaps the data describe direct variation with a power of depth. If we try the equation \(L = kd^2\) and use the point \((1, 30)\), we find that \(30 = k · 1^2\), so \(k = 30\). You can check that the equation \(L = 30d^2\) does fit the rest of the data points.
