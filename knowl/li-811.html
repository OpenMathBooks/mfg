First evaluate the function \(f\) for \(x = 2\): \begin{equation*}f(\alert{2}) = \alert{2}^3 + 2 = \blert{10}\end{equation*} Then evaluate the inverse function \(f^{−1}\) at \(y = 10\): \begin{equation*}f^{−1}(\blert{10}) = \sqrt[3]{\blert{10} − 2} = \sqrt[3]{8}= \alert{2}\end{equation*} We started and ended with \(2\).
