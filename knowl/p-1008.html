In Example 5.1.5, you can check that \(f(2) = 10\). In fact, the two statements \begin{equation*}f^{−1}(10) = 2 ~~\text{ and } ~~ f(2) = 10\end{equation*} are equivalent; they convey the same information. This fact is a restatement of our earlier observation about inverse functions, this time using inverse function notation.
