These properties do not mention subtraction or division. But we can define subtraction and division in terms of addition and multiplication. For example, we can define the difference \(a-b\) as follows: \begin{equation*} a − b = a + (−b) \end{equation*} where \(-b\), the additive inverse (or opposite) of \(b\), is the number that satisfies \begin{equation*}b + (−b) = 0\end{equation*} Similarly, we can define the quotient \(\frac{a}{b}\): \begin{equation*}\frac{a}{b}=a\left(\frac{1}{b} \right) \hphantom{blank} (b\ne 0) \end{equation*} where \(\frac{1}{b} \), the multiplicative inverse (or reciprocal) of \(b\), is the number that satisfies \begin{equation*}b\cdot\frac{1}{b}=1 \hphantom{blank} (b\ne 0)\end{equation*}
