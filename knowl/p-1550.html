If \(k\) is negative, then \(e^k\) is a fraction less than \(1\). For example, if \(k = −2\), \begin{equation*}e^{−2} = \frac{1}{e^2} \approx \frac{1}{7.3891} \approx 0.1353\end{equation*} Thus, for negative values of \(k\), the function \(P(t) = P_0 e^{kt}\) describes exponential decay.
