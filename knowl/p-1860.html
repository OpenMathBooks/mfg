We can graph this system in the standard window by solving each equation for \(y\). Enter \begin{align*} Y_1\amp = (21.06 − 3X)/−2.8 \\ Y_2\amp = (5.3 − 2X)/1.2 \end{align*} and then press ZOOM \(6\). (Do not forget the parentheses around the numerator of each expression.) Trace along the first line to find the intersection point. It appears to be at \(x=4.468051\), \(y =-2.734195\), as shown in Figure 8.6a. However, if we press the up or down arrow to read the coordinates off the second line, we see that for the same \(x\)-coordinate we obtain a different \(y\)-coordinate, as in Figure 8.6b. The different \(y\)-coordinates indicate that we have not found an intersection point, although we are close. However, the intersect feature can give us a better estimate, \(x = 4.36\), \(y = −2.85\). We can substitute these values into the original system to check that they satisfy both equations. \begin{align*} 3(\alert{4.36}) − 2.8(\blert{−2.85}) \amp = 21.06 \\ 2(\alert{4.36}) + 1.2(\blert{−2.85}) \amp = 5.3 \end{align*}
