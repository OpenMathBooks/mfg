The range of the function is the set of all \(f (x)\)-values that appear on the graph. We can see in Figure 1.6.8 that the lowest point on the graph is \((0, −6)\), so the smallest \(f (x)\)-value is \(−6\). The highest point on the graph is \((4, 10)\), so the largest \(f (x)\)-value is \(10\). Thus, the range of the function \(f\) is the interval \([−6, 10]\).
