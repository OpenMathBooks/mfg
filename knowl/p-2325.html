We first plot the given point, \((1, −4)\), as shown in Figure 1.137. Then we use the slope to find another point on the line. The slope is \begin{equation*}m =\frac{−3}{4}= \frac{\Delta y}{\Delta x}\end{equation*} so we move down \(3\) units and then \(4\) units to the right, starting from \((1, −4)\). This brings us to the point \((5, −7)\). We can then draw the line through these two points.
