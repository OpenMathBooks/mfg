Now consider the power and root operations in the opposite order; is it true that \(\sqrt[n]{a^n}=a \)? If the index \(n\) is an odd number, then the statement is always true. For example, \begin{equation*} \sqrt[3]{2^3}=\sqrt[3]{8}=2 ~~~\text{ and }~~~\sqrt[3]{(-2)^3}=\sqrt[3]{-8}=-2 \end{equation*} However, if \(n\) is even, we must be careful. Recall that the principal root \(\sqrt[n]{x} \) is always positive, so if \(a\) is a negative number, it cannot be true that \(\sqrt[n]{a^n}=a \). For example, if \(a = âˆ’3\), then \begin{equation*} \sqrt{(-3)^2}=\sqrt{9}=3 \end{equation*} Instead, we see that, for even roots, \(\sqrt[n]{a^n}=\abs{a} \).
