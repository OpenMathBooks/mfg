Addition:

If \(a = b\) and \(c\) is any number, then \(a + c = b + c\).

Subtraction:

If \(a = b\) and \(c\) is any number, then \(a - c = b - c\).

Multiplication

If \(a = b\) and \(c\) is any number, then \(a\cdot c = b\cdot c\).

Division

If \(a = b\) and \(c\) is any nonzero number, then \(\frac{a}{c} =\frac{b}{c} \).
