A logarithmic equation is one in which the variable appears inside of a logarithm. For example, \begin{equation*}\log_4 x = 3\end{equation*} is a log equation. To solve a log equation, remember that logarithms and exponentials with the same base are inverse functions. Therefore, \begin{equation*}y = \log_b x ~~ \text{ if and only if } ~~ x = b^y\end{equation*} Thus, we can rewrite a logarithmic equation in exponential form.
