We can also describe this measurement error, or error tolerance, using an absolute value inequality. Because the measured mass m can be no more than \(0.05\) from \(6\), we write \begin{equation*}\abs{m − 6} ≤ 0.05\end{equation*} Note that the solution of this inequality is \(5.95 \le m \le 6.05\).
