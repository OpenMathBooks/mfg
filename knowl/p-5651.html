The equation \(x^6 − 4x^3 − 5 = 0\) is not quadratic, but if we make the substitution \(u = x^3\), the equation becomes \(u^2 − 4u − 5 = 0\). An equation is called quadratic in form if we can use a substitution to write it as \(au^2 + bu + c = 0\), where \(u\) stands for an algebraic expression. Such equations can be solved by the same techniques we use to solve quadratic equations.
