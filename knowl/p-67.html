Use matrix reduction to solve the system \begin{align*} x + 3z \amp = −11 \\ 2x + y + z \amp = 1 \\ −3x − 2y = 3 \end{align*} Follow the suggested steps:

  1. Write the augmented matrix for the system.

  2. Add \(-2\)(row 1) to (row 2).

  3. Add \(3\)(row 1) to (row 3).

  4. Add \(2\)(row 2) to (row 3).

  5. Add \(2\)(row 2) to (row 3).
