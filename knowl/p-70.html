You already know that \(\sqrt{−9}\) is not a real number, because there is no real number whose square is \(−9\). Similarly, \(\sqrt[4]{−16}\) is not a real number, because there is no real number \(r\) for which \(r^4 = −16\). (Both of these radicals are complex numbers. Complex numbers are discussed in Chapter 7.) In general, we cannot find an even root (square root, fourth root, and so on) of a negative number.
