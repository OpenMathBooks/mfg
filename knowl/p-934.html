It is important to check the solution to a radical equation, because it is possible to introduce false, or extraneous, solutions when we square both sides of the equation. For example, the equation \begin{equation*}\sqrt{x}=-5 \end{equation*} has no solution, because \(\sqrt{x}\) is never a negative number. However, if we try to solve the equation by squaring both sides, we find \begin{align*} (\sqrt{x})^\alert{2}\amp = (-5)^\alert{2} \\ x\amp = 25 \end{align*} You can check that \(25\) is not a solution to the original equation, \(\sqrt{x}=-5 \), because \(\sqrt{25} \) does not equal \(-5\).
