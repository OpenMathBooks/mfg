Although the same symbol, \(^{−1}\), is used for both reciprocals and inverse functions, the two notions are not equivalent. That is, the inverse of a given function is usually not the same as the reciprocal of that function. In Example 5.1.6, note that \(f^{−1}(y)\) is not the same as the reciprocal of \(f (y)\), because \begin{equation*}\frac{1}{f(y)}= \frac{1}{y^3+ 2} ~~\text{ but }~~ f^{−1}(y)= \sqrt[3]{y − 2}\end{equation*} To avoid confusion, we use the notation \(\frac{1}{f}\) to refer to the reciprocal of the function \(f\).
