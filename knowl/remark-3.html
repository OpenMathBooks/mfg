Remark1.35Making a Table of Values with a Calculator

We can use a graphing calculator to make a table of values for a function defined by an equation. For the function in Example 1.33, \begin{equation*}h = 1776 − 16t^2\end{equation*} we begin by entering the equation: Press the Y= key, clear out any other equations, and define \(Y_1 = 1776 − 16X^2.\)

Next, we choose the \(x\)-values for the table. Press 2ndWINDOW to access the TblSet (Table Setup) menu and set it to look like Figure 1.36. This setting will give us an initial x-value of 0 \((TblStart = 0)\) and an increment of one unit in the x-values, \((\Delta Tbl = 1)\). It also fills in values of both variables automatically. Now press 2nd GRAPH to see the table of values, as shown in Figure 1.37. From this table, we can check the heights we found in Example 1.33. Now try making a table of values with \(TblStart = 0\) and \(\Delta Tbl = 0.5\). Use the ↑ and ↓ arrow keys to scroll up and down the table.
