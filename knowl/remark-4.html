Remark1.2.10Using a Calculator to Graph a Function

We can also use a graphing calculator to obtain a table and graph for the function in Example 1.2.8. We graph a function just as we graphed an equation. For this function, we enter

\begin{equation*}Y_1 = \sqrt{~^~}(X+4)\end{equation*}
and press \(6\) for the standard window. (See <<appendix-b>> for details.) The calculator’s graph is shown in Figure 1.2.11.
