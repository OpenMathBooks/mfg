Figure 1.128 shows the graph of the function \(h\) in Example 1.126 with the domain values marked on the horizontal axis and the range values marked on the vertical axis. Imagine a rectangle whose length and width are determined by those segments, as shown in Figure 1.128. All the points \((v, h(v))\) on the graph of the function lie within this rectangle. This rectangle is a convenient window in the plane for viewing the function. Of course, if the domain or range of the function is an infinite interval, we can never include the whole graph within a viewing rectangle and must be satisfied with studying only the important parts of the graph.
