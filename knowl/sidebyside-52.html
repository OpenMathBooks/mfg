Graphs can be helpful for working with absolute values. Consider the simple equation \(\abs{x} = 5\), which has two solutions, \(x = 5\) and \(x = −5\). In fact, we can see from the graph in Figure 2.106 that the equation \(\abs{x} = k\) has two solutions if \(k \gt 0\), one solution if \(k = 0\), and no solution if \(k \lt 0\).
