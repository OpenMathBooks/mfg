Solution5.4.6.1

We actually plot the logs of the values, so we first compute the base \(10\) logarithm of each number.

Then we plot each logarithm, estimating its position between integer exponents. For example, we plot the first value, \(−3.15\), closer to \(−3\) than to \(−4\). The finished plot is shown in Figure 5.4.7.
