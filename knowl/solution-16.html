Solution 1.3.8.1

Choose several convenient values for \(x\) and evaluate the function to find the corresponding \(f(x)\)-values. For this function we cannot choose \(x\)-values less than \(−4\), because the square root of a negative number is not a real number. \[f(\alert{−4}) =\sqrt{\alert{−4} + 4}=\sqrt{0}= 0\] \[f(\alert{−3}) =\sqrt{\alert{−3} + 4}=\sqrt{1}= 1\] \[f(\alert{0}) =\sqrt{\alert{0} + 4}=\sqrt{4}=2\] \[f(\alert{2}) =\sqrt{\alert{2} + 4}=\sqrt{6}\approx 2.45\] \[f(\alert{5}) =\sqrt{\alert{5} + 4}=\sqrt{9}=3\] The results are shown in the table.
