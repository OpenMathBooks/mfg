Solution6.56.1

We first write the equation in standard form as a quadratic equation in the variable \(x\text{.}\)

\begin{equation*} x^2 - yx + (y - 2) = 0 \end{equation*}

Expressions in \(y\) are treated as constants with respect to \(x\text{,}\) so that \(a = 1\text{,}\) \(b = -y\text{,}\) and \(c = y -2\text{.}\) Substitute these expressions into the quadratic formula. \begin{align*} x \amp=\frac{-(-y) \pm\sqrt{(-y)^2 - 4(1)(y - 2)}}{2(1)}\\ \amp = \frac{y \pm\sqrt{y^2 - 4y + 8}}{2} \end{align*}
