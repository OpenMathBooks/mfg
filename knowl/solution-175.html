Solution6.53.1

For this equation, \(a = −2\), \(b = 1\), and \(c = 1\). The \(x\)-coordinate of the vertex is given by \begin{equation*}x_v =\frac{−b}{2a}=\frac{-1}{2(-2)}=\frac{1}{4}\end{equation*} To find the \(y\)-coordinate of the vertex, we substitute \(x = \alert{\frac{1}{4}}\) into the equation. We can do this by hand to find \begin{align*} y_v\amp= −2\left(\alert{\frac{1}{4}}\right)^2+ \alert{\frac{1}{4}}+1\\ \amp= −2\left(\frac{1}{16}\right)+ \frac{4}{16}+\frac{16}{16}=\frac{18}{16} =\frac{9}{8} \end{align*} So the coordinates of the vertex are \(\left(\frac{1}{4},\frac{9}{8}\right)\). Alternatively, we can use the calculator to evaluate \(−2x^2 + x + 1\) for \(x = 0.25\). The calculator returns the \(y\)-value \(1.125\). Thus, the vertex is the point \((0.25, 1.125)\), which is the decimal equivalent of \(\left(\frac{1}{4},\frac{9}{8}\right)\).
