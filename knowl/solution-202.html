Solution7.2.3.1

The graph of the basic cubic is shown in Figure 7.2.4a. To help us understand the graph of the polynomial \(P(x) = x^3 - 4x\text{,}\) we will evaluate the function to make a table of values. We can do this by hand or use the Table feature on the graphing calculator.

The graph of \(P(x) = x^3 - 4x\) is shown in Figure 7.2.4b. It is not exactly the same shape as the basic cubic—it has two turning points—but it is similar, especially at the edges of the graphs.
