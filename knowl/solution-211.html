Solution7.3.17.1

We substitute \(x = 1 + i\) to find \begin{align*} f (1 + i ) \amp = (1 + i )^2 − 2(1 + i ) + 2 \\ \amp = 1^2 + 2i + i^2 − 2 − 2i + 2 \\ \amp = 1 + 2i + (−1) − 2 − 2i + 2 \\ \amp =0 \end{align*} Thus, \(f(1 + i ) = 0\), so \(1 + i\) is a solution of \(x^2 − 2x + 2 = 0\).
