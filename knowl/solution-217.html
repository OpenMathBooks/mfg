Solution7.4.7.1

The domain of \(f\) is the set of all real numbers except \(3\), because the denominator, \((x − 3)^2\), equals \(0\) when \(x = 3\). The domain of \(g\) is the set of all real numbers except \(−1\), because \(x + 1\) equals zero when \(x=-1\). The denominator of the function \(h\), \(x^2 + 4\), is never equal to zero, so the domain of \(h\) is all the real numbers. The domain of \(k\) is the set of all real numbers except \(3\) and \(−3\), because \(x^2 − 9\) equals \(0\) when \(x = 3\) or \(x = −3\).
