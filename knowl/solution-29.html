Solution1.4.28.1
  1. We set \(x_1 = \alert{1}\) and \(x_2 = \alert{4}\) and find the function values at each point. \begin{equation*}f (x_1) = f (\alert{1}) = \alert{1}^2 − 6(\alert{1}) = −5\end{equation*}\begin{equation*}f (x_2) = f (\alert{4}) = \alert{4}^2 − 6(\alert{4}) = −8\end{equation*} Then \begin{equation*}m = \frac{f (x_2) − f (x_1)}{x_2 − x_1} =\frac{−8 − (−5)}{4 − 1}=\frac{−3}{3}= −1\end{equation*}
  2. We set \(x_1 = \alert{2}\) and \(x_2 = \alert{5}\) and find the function values at each point. \begin{equation*}f (x_1) = f (\alert{2}) = \alert{2}^2 − 6(\alert{2}) = −8\end{equation*}\begin{equation*}f (x_2) = f (\alert{5}) = \alert{5}^2 − 6(\alert{5}) = −5\end{equation*} Then \begin{equation*}m = \frac{f (x_2) − f (x_1)}{x_2 − x_1} =\frac{−5 − (−8)}{5 − 2}=\frac{3}{3}= 1\end{equation*}
