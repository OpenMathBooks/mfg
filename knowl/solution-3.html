Solution1.7.1

The domain of \(f\) is the set of all real numbers except \(3\text{,}\) because the denominator, \((x − 3)^2\text{,}\) equals \(0\) when \(x = 3\text{.}\) The domain of \(g\) is the set of all real numbers except \(−1\text{,}\) because \(x + 1\) equals zero when \(x=-1\text{.}\) The denominator of the function \(h\text{,}\) \(x^2 + 4\text{,}\) is never equal to zero, so the domain of \(h\) is all the real numbers. The domain of \(k\) is the set of all real numbers except \(3\) and \(−3\text{,}\) because \(x^2 − 9\) equals \(0\) when \(x = 3\) or \(x = −3\text{.}\)
