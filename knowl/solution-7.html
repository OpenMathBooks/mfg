Solution1.1.18.1

  1. To find the \(x\)-intercept, set \(y = \alert{0}\).

    \begin{align*} 150x-18(\alert{0})&=9000 &&\text{Simpify.}\\ 150x&=9000 &&\text{Divide both sides by 150.}\\ x&=60 && \end{align*}

    The \(x\)-intercept is the point \((60, 0)\). To find the \(y\)-intercept, set \(x = \alert{0}\).

    \begin{align*} 150(\alert{0})-18y&=9000 &&\text{Simpify.}\\ -180y&=9000 &&\text{Divide both sides by } -180\text{.}\\ y&=-50 && \end{align*}

    The \(y\)-intercept is the point \((0, −50)\).

  2. Scale both axes in intervals of 10 and then plot the two intercepts, \((60, 0)\) and \((0, −50)\). Draw the line through them, as shown in Figure 1.1.19. Now find another point and check that it lies on this line. We choose \(x = \alert{20}\) and solve for \(y\). \begin{gather*} 150(\alert{20}) −180y = 9000\\ 3000 −180y = 9000\\ −180y = 6000\\ y =−33.\overline{3} \end{gather*} Plot the point \((20, −33\frac{1}{3})\). Because this point lies on the line, we can be reasonably confident that our graph is correct.
