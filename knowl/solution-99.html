Solution4.2.4.1

Evaluate each function for several convenient values, as shown in Table 4.2.5.

Plot the points for each function and connect them with smooth curves. For positive \(x\)-values, \(g(x)\) is always larger than \(f (x)\), and is increasing more rapidly. In Figure 4.2.6, \(g(x) = 4^x\) climbs more rapidly than \(f (x) = 3^x\). Both graphs cross the \(y\)-axis at (0, 1).
