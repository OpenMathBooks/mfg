Subsubsection1.1.3STUDY QUESTIONS

  1. The properties of logs are really another form of which familiar laws?

  2. Which log property allows us to solve an exponential equation whose base is not \(10\)?

  3. Explain why \(12\cdot 10^{3x}\) is not the same as \(120^{3x}\).

  4. Which of the following expressions are equivalent? \begin{equation*}\log\left(\dfrac{x}{4} \right) ~~~ \dfrac{\log x}{\log 4} ~~~ \log(x-4) ~~~ \log x - \log 4 \end{equation*}

  5. Which of the following expressions are equivalent? \begin{equation*}\log(x+2) \right) ~~~ \log x+\log 2 ~~~ \log(2x) ~~~ (\log 2) (\log x) \end{equation*}

  6. Which of the following expressions are equivalent? \begin{equation*}\log x^3 ~~~(\log 3)(\log x) ~~~ 3\logx ~~~ \log x - \log 3^x \end{equation*}
